# **App Name**: Shelby Drive

## Core Features:

- Wallet Connect: A 'Connect Wallet' button to allow users to link their Petra wallet for Aptos blockchain authentication and transaction signing.
- Secure File Upload: An intuitive drag-and-drop or file selection interface for uploading files to Shelby Protocol. This will utilize the @shelby-protocol/sdk for automatic data chunking and erasure coding to ensure redundancy and enterprise-grade reliability.
- Upload Progress & Notifications: Real-time display of upload progress and clear notifications upon successful completion or failure.
- Data Retrieval & Link Sharing: After a successful upload, display a unique ID or direct link for users to quickly retrieve their file from Shelby's distributed storage nodes.
- Error Handling & Feedback: Robust error handling for scenarios like a disconnected wallet or failed uploads to the global Shelby network, providing user-friendly feedback.
- Intelligent File Tagging: An AI-powered tool that analyzes the content or type of an uploaded file and suggests relevant tags or categories to improve organization and searchability.

## Style Guidelines:

- Color Palette (Dark Scheme): Primary accent blue-violet (#2626D9) for key actions and branding. A subtle, desaturated dark background (#16161D) for a modern, tech-centric feel. A bright, electric blue (#55A6F6) for secondary accents and interactive elements to create visual dynamism.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, tech-forward, and bold appearance. Body font: 'Inter' (sans-serif) for clean readability and neutrality across various content types.
- Utilize modern, clean, and geometric icons that align with a digital and Web3 aesthetic. Icons should clearly represent actions like 'Connect Wallet', 'Upload', 'Download', and 'Share'.
- A clean, spacious, and responsive layout emphasizing the speed, censorship resistance, and global reach of Shelby Protocol. Focus on centralizing the file upload and retrieval functions for an intuitive user flow.
- Subtle and fluid animations for user interactions, such as wallet connection feedback, upload progress indicators, and state transitions, to enhance the perception of speed and responsiveness.
